# VoteSmart India – Election Assistant Portal

VoteSmart India is a comprehensive web application designed to guide Indian citizens through the electoral process. It features a Gemini-powered AI assistant (KIRAN), multi-language support (10+ languages), and a streamlined voter registration journey.

## 🚀 Live Links
- **Production Website:** [https://votesmart-c922a.web.app](https://votesmart-c922a.web.app)
- **Backend API:** [https://votesmart-backend-656894301486.asia-south1.run.app](https://votesmart-backend-656894301486.asia-south1.run.app)

## ✨ Features
- **AI Assistant (KIRAN)**: Interactive guide for electoral queries.
- **Voter Journey**: Step-by-step eligibility checker and registration guide.
- **Multilingual**: Supports English, Hindi, Bengali, Telugu, Marathi, Tamil, Gujarati, Kannada, Odia, Malayalam, and Punjabi.
- **Admin Dashboard**: For managing registrations and feedback.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Framer Motion, Tailwind-ready Vanilla CSS.
- **Backend**: FastAPI (Python), SQLAlchemy, PostgreSQL.
- **Hosting**: Firebase Hosting (Frontend), Google Cloud Run (Backend).
- **AI**: Google Gemini Pro API.

## 📁 Project Structure
- `/frontend`: React application.
- `/backend`: FastAPI application and Docker configuration.

## 🔧 Local Development

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn app.main:app --reload`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
